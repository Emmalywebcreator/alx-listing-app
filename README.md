# ALX Listing App
This project was scaffold using 

npx create-next-app@latest alx-listing-app --typescript --tailwind --eslint --no-app-router --no-src-dir

A foundational scaffold for building a modern Airbnb-style listing page using Next.js, TypeScript, TailwindCSS, and ESLint.

## Folder Structure
- components/ — Reusable UI components
- interfaces/ — TypeScript interface definitions
- constants/ — Shared static values
- public/assets — Project images & SVGs

## How to Run
npm install
npm run dev
Open http://localhost:3000
